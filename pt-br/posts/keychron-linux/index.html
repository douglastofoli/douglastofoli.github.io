Keychron Linux Teclas de Função

Quando eu comprei um Keychron K2 eu sabia que ele funcionava no Windows e Mac, mas eu uso Linux. Porém é apenas um teclado… ele deve funcionar.

Bem… E funcionou! Mas no meu SO Arch Linux com Gnome teve um pequeno problema. As teclas de função (fn) não funcionavam perfeitamente. Na verdade as teclas FN estavam definidas para MacOS, não Windows.

Como eu corrigi:

  • Altere o switch lateral do teclado para o modo Windows
  • Segure Fn + X + L por 4 segundos para ativar a linha de teclas de função

Digite no terminal:

echo 0 | sudo tee /sys/module/hid_apple/parameters/fnmode

Fazendo isso seu teclado deverá funcionar perfeitamente.

Com um pequeno detalhe: para persistir as mudanças você precisa adicionar uma opção ao módulo hid_apple

echo "options hid_apple fnmode=0" | sudo tee -a /etc/modprobe.d/hid_apple.conf

Importante!

Agora você precisa fazer o rebuild do seu initramfs se o módulo hid_apple foi incluído.

  • Ubuntu: sudo update-initramfs -u
  • Arch: mkinitcpio -P

Pronto! Após reiniciar seu SO não irá mais perder essa configuração e suas teclas de função irão funcionar assim como no Windows.
