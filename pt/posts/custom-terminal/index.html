Seu terminal com Oh My Zsh, Spaceship, Dracula e mais funcionalidades

Deixando seu terminal melhor para o trabalho

Olá. Vamos configurar um terminal bonito e produtivo. Esse não é um tutorial bem detalhado. Eu vou deixar aqui links e alguns comandos de como eu configurei meu terminal. É fácil, basta seguir o passo a passo na ordem.

Terminator

Passo a passo

1. Instale Zsh

Você precisa install um shell diferente. Mas calma. Nada vai mudar em relação ao bash.

Acessa esta página. Isso vai ajudar você a instalar o Zsh de acordo com sua distro Linux.


2. Instale Oh My Zsh

Para o próximo passo install o Oh My Zsh. Ele vai permitir mais customizações para o Zsh.

Execute no terminal:

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

3. Tema Dracula

Eu escolhi o tema Dracula como meu dark theme. Você pode escolher um tema de sua preferência.
Tema Dracula para Terminator


4. Tema shell Spaceship

Spaceship é fácil de instalar. Apenas execute no terminal:

git clone https://github.com/spaceship-prompt/spaceship-prompt.git "$ZSH_CUSTOM/themes/spaceship-prompt" --depth=1

E crie um link simbólico para seu diretório de temas padrão:

ln -s "$ZSH_CUSTOM/themes/spaceship-prompt/spaceship.zsh-theme" "$ZSH_CUSTOM/themes/spaceship.zsh-theme"

Ative o Spaceship como tema principal. Edite ~/.zshrc e encontre por ZSH_THEME

ZSH_THEME="spaceship"


Para mais sobre Spaceship: GitHub - Spaceship


5. Configurações adicionais

Ao final do arquivo ~/.zshrc adicione o conteúdo abaixo:

SPACESHIP_PROMPT_ORDER=(
  user          # Username section
  dir           # Current directory section
  host          # Hostname section
  git           # Git section (git_branch + git_status)
  node
  rust
  docker
  hg            # Mercurial section (hg_branch  + hg_status)
  exec_time     # Execution time
  line_sep      # Line break
  vi_mode       # Vi-mode indicator
  jobs          # Background jobs indicator
  exit_code     # Exit code section
  char          # Prompt character
)
SPACESHIP_USER_SHOW=always
SPACESHIP_PROMPT_ADD_NEWLINE=false
SPACESHIP_CHAR_SYMBOL="❯"
SPACESHIP_CHAR_SUFFIX=" "

SPACESHIP_RUST_SYMBOL="🦀  "
SPACESHIP_RUST_COLOR=red

Você pode personalizar tudo! Veja a documentação no passo anterior.


6. Plugins

Finalmente nós vamos instalar alguns plugins. Para o gerenciador de plugins iremos usar um fork do Zinit. Para instalar execute no terminal:

sh -c "$(curl -fsSL https://git.io/zinit-install)"

Com a instalação completa, edite ~/.zshrc e adicione o conteúdo abaixo no final do arquivo:

zinit light zdharma-continuum/fast-syntax-highlighting
zinit light zsh-users/zsh-autosuggestions
zinit light zsh-users/zsh-completions

Salve o arquivo, feche o terminal e abra novamente. Isso irá instalar três plugins. Eles fazem:

  • destaque de cor
  • sugestão de comandos
  • completa comandos

7. Terminal VSCode [Bonus]

Para ativar e usar esse novo terminal no VSCode edite seu arquivo settings.json e adicione esta instrução:

"terminal.integrated.defaultProfile.linux": "zsh"


E fim

Curtiu esse terminal? Limpo, rápido e produtivo. Você pode usar a área de comentários para sugerir dicas e screenshots do seu novo terminal.
