Iniciando um novo projeto [planejamento]

O planejamento:

Eu iniciei um novo projeto. Ainda sem nome, mas eu já sei sobre o que vai ser. Parece um pouco bobo mas eu quero fazer um gerenciador de dinheiro (Money Manager). Uma aplicação que você pode adicionar registros para despesas e receitas.

Como eu iniciei? Planejamento! Eu estou planejando um banco de dados. Eu pretendo usar PostgreSQL como meu banco relacional.

Eu estou usando dbdiagram.io para criar minhas tabelas

dbdiagram.io

No lado esquerdo eu escrevi em código o que eu quero representar nas minhas tabelas. O resto da tela é uma representação gráfica das tabelas.

Muitas referências:

Sempre que você ver “_id” em uma coluna, é uma referência para outra tabela. O nome que vem antes é o nome da outra tabela.

A represenção em código é:

Ref: accounts.user_id > users.id

Tabela.Chave Estrangeira > Outra Tabela.Chave Primária

Tecnologias:

Eu quero usar nesse projeto:

  • Elixir para backend
  • GraphQl para API
  • PostgreSQL para banco de dados
  • ReactJS para frontend
  • […] para Mobile

Bora!

Esse é o primeiro post de uma saga. Eu quero documentar todos os status do projeto aqui e explicar detalhes sobre cada processo executado. Quando o projeto alcançar maturidade irei deixar disponível no GitHub para visualização e contribuição. Vamos trabalhar e fazer algo diferente acontecer.

Até logo!
