Keychron Linux Function Keys

When I bought a Keychron K2 I knew it would work in Windows and Mac, but I use Linux. But it’s just a keyboard… it should work.

Well… It works! But on my OS Arch Linux with Gnome had a little problem. The FN keys didn’t work perfectly. Actually the FN keys were for MacOS, not Windows.

How I fixed:

  • Use the switch on side of keyboard to change to Windows mode
  • Hold Fn + X + L for 4 seconds to set the key row to Function mode

Type in the terminal:

echo 0 | sudo tee /sys/module/hid_apple/parameters/fnmode

When doing this your keyboard should work perfectly.

With a small detail: to persist this change you need to add a module option for hid_apple

echo "options hid_apple fnmode=0" | sudo tee -a /etc/modprobe.d/hid_apple.conf

Important!

Now you need to rebuild your initramfs if hid_apple module is included.

  • Ubuntu: sudo update-initramfs -u
  • Arch: mkinitcpio -P

Good! After reboot your OS don’t will go miss this configuration and your FN keys will work as if it were Windows.
