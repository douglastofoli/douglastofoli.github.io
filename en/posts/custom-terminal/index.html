Your terminal with Oh My Zsh, Spaceship, Dracula and more functionalities

Letting your terminal better to work

Hello there. Let’s configuration a beauty and productive terminal. This is not a detailed tutorial. I will leave here links and some commands on how I set up my terminal. It’s easy, just follow in order

Terminator

Step by step

1. Install Zsh

You need to install a different shell. But calm. Nothing will change in relation to bash

Access this page. This will help you with installation for your Linux distro.


2. Install Oh My Zsh

To next step install Oh My Zsh. It will let the Zsh more customizable.

Run on terminal:

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

3. Dracula Theme

I did choose Dracula for my dark theme. You can choose a theme of your preference.
Dark theme for Terminator


4. Theme shell Spaceship

Spaceship is easy to install. Just run on terminal:

git clone https://github.com/spaceship-prompt/spaceship-prompt.git "$ZSH_CUSTOM/themes/spaceship-prompt" --depth=1

And create symlink to your custom theme directory:

ln -s "$ZSH_CUSTOM/themes/spaceship-prompt/spaceship.zsh-theme" "$ZSH_CUSTOM/themes/spaceship.zsh-theme"

Active Spaceship to your major theme. Edit ~/.zshrc and find ZSH_THEME

ZSH_THEME="spaceship"


For more about Spaceship: GitHub - Spaceship


5. Aditional Configurations

In the end of file ~/.zshrc add the content below:

SPACESHIP_PROMPT_ORDER=(
  user          # Username section
  dir           # Current directory section
  host          # Hostname section
  git           # Git section (git_branch + git_status)
  node
  rust
  docker
  hg            # Mercurial section (hg_branch  + hg_status)
  exec_time     # Execution time
  line_sep      # Line break
  vi_mode       # Vi-mode indicator
  jobs          # Background jobs indicator
  exit_code     # Exit code section
  char          # Prompt character
)
SPACESHIP_USER_SHOW=always
SPACESHIP_PROMPT_ADD_NEWLINE=false
SPACESHIP_CHAR_SYMBOL="❯"
SPACESHIP_CHAR_SUFFIX=" "

SPACESHIP_RUST_SYMBOL="🦀  "
SPACESHIP_RUST_COLOR=red

You can customize everything! See the documentation in the previous step.


6. Plugins

Finally we go to install some plugins. To manage the plugins we will use a Zinit fork. To install run on terminal:

sh -c "$(curl -fsSL https://git.io/zinit-install)"

With the installation completed, edit ~/.zshrc and add the content below in end of file:

zinit light zdharma-continuum/fast-syntax-highlighting
zinit light zsh-users/zsh-autosuggestions
zinit light zsh-users/zsh-completions

Save the file, close the terminal and open again. It install the three plugins. That do:

  • color highlight
  • autosuggestions of commands
  • autocomplete improved

7. Terminal VSCode [Bonus]

To activate and use this new terminal in VSCode edit your settings.json and add this instruction:

"terminal.integrated.defaultProfile.linux": "zsh"


The end
