Starting New Project [Planning]

The Planning:

I started a new project. Unnamed yet, but I already know about what will be. It looks a bit silly but I want to do a money manager. An application that you can add entries for expenses and incomes.

How I started? Planning! I am planning the database. I intend to use PostgreSQL for my database.

I am using dbdiagram.io to create my schemas

dbdiagram.io

On the left side I wrote in code what I want to represent in my tables. The rest of the screen is a graphical representation of the tables.

Very much references:

Ever you see “_id” on column is a reference to another table. The name that comes before is the name of the other table.

The representation in code is:

Ref: accounts.user_id > users.id

Table.Foreign Key > Other Table.Primary Key
